# 插件

您可以在Plugins for GitBook查找适合自己的插件和用法，多数插件是针对书籍模式。

# 默认插件

  • highlight：高亮文本
  • search：为您的书添加了一个交互式搜索栏
  • sharing：右上角的分享按钮
  • fontsetting：添加字体、风格和背景设置按钮
  • lunr：为搜索插件提供了一个后端

# 常用插件

showLevel: boolean 标题是否显示层级序号，页面标题和导航中的标题都会加上层级显示。

# 其他插件

插件太多影响速度，够用就好。参考插件 · GitBook 使用教程，整理使用插件如下。

如何开发插件，请看：Plugins · GitBook Toolchain Documentation

# 代码块

  • Prsim - 使用 Prism.js 高亮代码
  • Ace Plugin - 支持ace
  • Include Codeblock - 用代码块显示包含文件的内容
  • Terminal - 模拟终端样式
  • Klipse - 集成 Kplise (online code evaluator)

# 图表

  • Mermaid-gb3 - 支持渲染 Mermaid 图表
  • Puml - 支持渲染 uml 图
  • Graph - 使用 function-plot 绘制数学函数图
  • Chart - 绘制图形
  • Include-csv - 显示 csv 文件内容

# 页面结构

  • Github - 添加github图标
  • Github Buttons - 添加项目在 Github 上的 star、fork、watch 信息
  • Splitter - 使侧边栏的宽度可以自由调节
  • Sharing-plus - 分享当前页面
  • Tbfed-pagefooter - 为页面添加页脚
  • Expandable-chapters-small - 使左侧的章节目录可以折叠
  • Simple-page-toc - 自动生成本页的目录结构
  • Anchors - 添加 Github 风格的锚点
  • Versions-select - 添加版本选择的下拉菜单
  • Rss - 添加 rss 订阅功能

# 文本

  • Advanced Emoji - 支持 emoji 表情
  • Emphasize - 为文字加上底色
  • Todo - 添加 Todo 功能
  • Exercises-文本中插入题目

# 其他

  • GA - Google 统计
  • 3-ba - 百度统计
  • Local Video - 使用 Video.js 播放本地视频
  • Musicxml - 支持 musicxml 格式的乐谱渲染

# GitBook 命令行工具

# 安装 GitBook 命令行工具

环境要求：NodeJS(v4.0.0及以上)

通过 npm 安装，运行下面的命令进行安装

npm install -g gitbook-cli
1

gitbook-cli是GitBook 的命令行工具，gitbook-cli 会将下载的 gitbook 的不同版本放到 ~/.gitbook中, 可以通过设置GITBOOK_DIR环境变量来指定另外的文件夹。

# 预览书籍

您可以在书籍根目录下，使用gitbook serve命令会运行一个服务器，通过http://localhost:4000/预览书籍。运行该命令后会在书籍的文件夹中生成一个 _book 文件夹，里面的内容是生成的 html 文件。您也可以使用gitbook build命令来生成网页而不开启服务器。

-o, --output <directory> 输出文件件, 默认为 ./_book
-f, --format <name> 产生的书籍的类型, 默认为静态站点, 可用的格式为: site, page, ebook, json
--config <config file> 配置文件, 默认为 book.js 或 book.json
1
2
3

您可以用此操作来调试book.json中的配置信息，调试成功后再 push 到远程仓库。增加环境变量 DEBUG=true 来得到更好的错误信息(包含错误堆栈)。

export DEBUG=true
gitbook build
1

# 其他命令

生成时指定gitbook的版本, 本地没有会先下载

gitbook build --gitbook=2.0.1
1

列出本地所有的gitbook版本

gitbook ls
1

列出远程可用的gitbook版本

gitbook ls-remote
1

安装对应的gitbook版本

gitbook fetch 标签/版本号
1

更新到gitbook的最新版本

gitbook update
1

卸载对应的gitbook版本

gitbook uninstall 2.0.1
1

指定log的级别

gitbook build --log=debug
1

输出错误信息

gitbook builid --debug
1
